1. Entendimento do Problema

O projeto Indicar Pneus é uma aplicação web Full-Stack cujo objetivo é:
Permitir cadastro de clientes
Permitir cadastro de veículos do cliente
Permitir cadastro de pneus que o cliente utiliza
Utilizar dados técnicos de veículos para indicar o pneu ideal

O sistema terá:
Back-end Python (FastAPI)
Front-end Web (Streamlit)
Banco de dados relacional (SQLite via SQLAlchemy)
CRUD de clientes e informações relacionadas.

Além disso:
O sistema deve ajudar o cliente a escolher pneus adequados, pois muitos usuários não sabem qual pneu comprar.

2.2 Dados de pneus
As dimensões seguem um padrão universal.

Exemplo:
205/55 R16 91V

Onde:
205 = largura
55 = perfil
R16 = aro
91 = índice de carga
V = velocidade

Logo é possível fazer regras de recomendação automatizadas.
